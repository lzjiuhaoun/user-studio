# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

---

## 项目概述

此项目是一个个人工作站，功能点为：记录用户填写的网站、配置网站说明，支持点击网站卡片跳转到对应的网站。

---

## 默认回答语言

所有解释性内容、步骤说明、分析、文字回答均默认使用 **中文**。
所有代码块 (code) 中的内容保持原始语言，不进行中文化或翻译。

---

## 强制要求

1. 大任务（执行时长超过20分钟）必须要拆分为小任务执行，任何任务都要分步骤追加写入 `task_list.md` 文件中，里面必须包括：
   - 主人的要求
   - 你的执行情况
   - 变动的文件
   - 代码

   不同的任务要用 `任务名+时间` 分隔，这个文件要能成为任何一个 agent 会话的上下文内容。

2. 我会把你生成的代码交给 Claude 和 Gemini agent 进行 review，他们会毫不留情地指出代码存在的低级错误，当错误数**多余10个**时，我会改用 Alibaba 的 qwen3.5 大模型。

3. 非必要不要引入依赖包，若必须要依赖包则需要将引入的依赖包写入 `dependencies.md` 依赖列表清单文件中。
