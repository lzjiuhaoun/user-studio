# WeTab 风格重设计 - 设计文档

## 概述

将现有「个人工作站」浏览器起始页改造为 WeTab 风格，采用毛玻璃效果、渐变背景、大图标卡片布局，提供更现代化的视觉体验。

## 需求确认

| 项目 | 决策 |
|------|------|
| 搜索栏 + 时间天气 | ✅ 保留搜索栏（仅搜索书签），时间组件（不需要天气） |
| 毛玻璃卡片效果 | ✅ 采用 glassmorphism 设计 |
| 大图标网格布局 | ✅ 卡片尺寸 180x140px，圆角 16px |
| 暗色 + 渐变背景 | ✅ 多色渐变背景（紫/蓝/粉） |
| 背景方案 | ✅ CSS 渐变，无需外部资源 |
| 分类管理 | ✅ 保留左侧边栏 |
| 搜索功能 | ✅ 仅快速访问书签 |
| 图标获取 | ✅ 默认 Google Favicon 代理，失败时支持手动上传 |

## 设计规格

### 1. 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│                      渐变背景 (#667eea → #764ba2 → #f093fb)   │
│  ┌─────────┐                                                  │
│  │ 侧边栏   │   ┌──────────────────────────────────────────┐  │
│  │         │   │  顶部区域                                  │  │
│  │ 分类列表 │   │  ┌──────────────────┐    ┌────────────┐   │  │
│  │ (毛玻璃) │   │  │    搜索栏        │    │  时间显示   │   │  │
│  │         │   │  └──────────────────┘    └────────────┘   │  │
│  │         │   └──────────────────────────────────────────┘  │
│  │         │   ┌──────────────────────────────────────────┐  │
│  │         │   │                                          │  │
│  │         │   │           网站卡片网格 (4-6列响应式)        │  │
│  │         │   │       (毛玻璃效果大卡片，悬停上浮)           │  │
│  │         │   │                                          │  │
│  └─────────┘   └──────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

### 2. 配色方案

```css
:root {
    /* 渐变背景 */
    --bg-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%);

    /* 毛玻璃 */
    --glass-bg: rgba(255, 255, 255, 0.1);
    --glass-bg-hover: rgba(255, 255, 255, 0.15);
    --glass-border: rgba(255, 255, 255, 0.2);
    --glass-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);

    /* 文字 */
    --text-primary: #ffffff;
    --text-muted: rgba(255, 255, 255, 0.7);

    /* 强调色 */
    --accent: #f093fb;
    --accent-hover: #f557c4;

    /* 圆角 */
    --radius-sm: 8px;
    --radius-md: 12px;
    --radius-lg: 16px;
    --radius-xl: 24px;

    /* 过渡 */
    --transition: 0.3s ease;
}
```

### 3. 组件规格

#### 3.1 侧边栏

- **宽度**: 240px（展开）/ 60px（折叠）
- **样式**: 毛玻璃背景 `backdrop-filter: blur(20px)`
- **内容**:
  - 标题「分类」（折叠时隐藏）
  - 分类列表：颜色指示点 + 名称 + 删除按钮（悬停显示）
  - 添加分类按钮
- **交互**: 可折叠，展开/收起动画 0.3s

#### 3.2 顶部区域

- **布局**: 搜索框居中 + 时间靠右
- **搜索框**:
  - 毛玻璃背景，圆角 24px
  - 宽度 400px（最大），占满可用空间
  - placeholder: "搜索书签..."
  - 输入时实时过滤下方卡片
- **时间显示**:
  - 数字时钟格式 HH:MM
  - 大字体 (2rem)，粗体

#### 3.3 网站卡片网格

- **布局**: CSS Grid，`repeat(auto-fill, minmax(160px, 1fr))`
- **间距**: 20px gap
- **卡片尺寸**: 最小 160px 宽，高度自适应（约 140px）
- **卡片样式**:
  - 毛玻璃背景
  - 圆角 16px
  - 内边距 20px
  - 边框 1px solid var(--glass-border)
- **卡片内容**:
  - 图标: 56x56px，圆角 12px，居中
  - 名称: 下方居中，最大2行，超出省略
- **悬停效果**:
  - `transform: translateY(-5px)`
  - 背景变亮
  - 阴影加深
  - 编辑/删除按钮浮现（右上角）

#### 3.4 欢迎/文件选择界面

- **样式**: 与整体风格一致，毛玻璃卡片居中
- **内容**: 标题 + 说明文字 + 两个按钮（选择文件/创建文件）

### 4. 交互规格

| 交互 | 行为 |
|------|------|
| 搜索框输入 | 实时过滤卡片，仅显示名称包含关键词的卡片 |
| 卡片点击 | 新窗口打开网站 URL |
| 卡片悬停 | 上浮 + 浮现操作按钮 |
| 编辑按钮 | 打开编辑模态框 |
| 删除按钮 | 确认后删除，网站转移至默认分类 |
| 添加按钮 | 打开添加模态框 |
| 分类点击 | 筛选显示该分类下的网站 |
| 侧边栏折叠 | 切换 sidebar.collapsed 类，动画过渡 |

### 5. 图标获取逻辑

```
1. 用户输入网址后，自动提取 domain
2. 尝试加载 Google Favicon: https://www.google.com/s2/favicons?domain={domain}&sz=64
3. 若加载失败（onerror），显示默认图标（一个灰色的地球图标）
4. 模态框中显示提示："图标加载失败，可手动上传"
5. 用户可选择上传自定义图标文件
```

### 6. 模态框样式

- **遮罩**: `rgba(0, 0, 0, 0.5)` + `backdrop-filter: blur(5px)`
- **内容框**: 毛玻璃背景，圆角 16px，最大宽度 420px
- **表单**: 垂直排列，标签在输入框上方
- **按钮**: 保存（accent色）/ 取消（透明边框）
