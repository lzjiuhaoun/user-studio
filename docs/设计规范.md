# WeTab 风格导航页 - 前端设计规范

## 1. 概述

- **设计风格**: WeTab 风格，玻璃拟态 (Glassmorphism)
- **主题**: 深色主题配背景图片
- **字体**: 无衬线字体 (font-sans)，时间显示使用等宽字体 (font-mono)
- **图标库**: Lucide Icons
- **样式框架**: Tailwind CSS (CDN)

---

## 7. 动效系统

### 7.1 过渡时长

| 类型 | 时长 |
|------|------|
| 默认过渡 | `duration-300` (300ms) |
| 慢速过渡 | `duration-500` (500ms) |

### 7.2 悬停效果

| 元素 | 效果 |
|------|------|
| 组件卡片 | `hover:-translate-y-1` 上移 4px |
| 图标按钮 | `hover:-translate-y-2` 上移 8px + `scale-110` |
| 添加按钮 | 图标旋转 90° |
| 时间显示 | `scale-105` 放大 5%，时长 500ms |

### 7.3 模糊效果

| 用途 | 值 |
|------|------|
| 背景遮罩 | `backdrop-blur-sm` (4px) |
| 组件背景 | `backdrop-blur-md` (12px) |

## 9. 组件规范

### 9.1 顶部工具栏

- 高度: `p-6` (24px)
- 布局: `flex justify-between items-center`
- 添加按钮: 胶囊形 `bg-white/10 backdrop-blur-md border border-white/20 rounded-full`
- 图标按钮: 圆形 `p-2 bg-white/10 rounded-full`

### 9.2 时钟组件

- 容器: `text-center mb-8`
- 时间: `font-mono font-bold tracking-wider` (加宽字间距)
- 日期: `mt-4 tracking-wide`
- 悬停: `hover:scale-105 transition-transform duration-500`
- 禁止选择: `select-none cursor-default`

### 9.3 搜索框

- 容器: `w-full max-w-2xl mb-12 relative group`
- 高度: `py-4` (16px 垂直内边距)
- 左侧图标: `left-6` 定位，`pl-14`
- 圆角: `rounded-full`
- 聚焦: `focus:ring-2 focus:ring-white/30`

### 9.4 玻璃卡片

```css
/* 标准玻璃卡片 */
bg-white/10 backdrop-blur-lg border border-white/20 rounded-3xl shadow-lg

/* 渐变玻璃卡片 */
bg-gradient-to-br from-white/15 to-white/5 backdrop-blur-lg
```

### 9.5 快捷方式图标

- 容器: `w-16 h-16 md:w-20 md:h-20` (64px → 80px)
- 圆角: `rounded-2xl md:rounded-3xl`
- 图标: `w-8 h-8`
- 标签: `mt-2 text-xs md:text-sm`

### 9.6 待办列表项

| 状态 | 样式 |
|------|------|
| 已完成 | `line-through text-white/50 bg-white/10` |
| 未完成 | `bg-white/10 border-white/50` |
| 普通 | `bg-white/5` |

复选框圆形: `w-4 h-4 rounded-full border`

### 9.7 底部名言

- 位置: `absolute bottom-6`
- 字号: `text-sm`
- 字重: `font-light`
- 颜色: `text-white/60 hover:text-white/90`
